# 翻訳差分チェック

引数で渡された日本語MDXファイルと対応する英語版ファイルを比較し、翻訳の差異を検出・報告する。

## 手順

1. 引数で渡された日本語MDXファイルを読み込む
2. パスの `ja/` を `en/` に置き換えて対応する英語ファイルを読み込む
3. 両ファイルの内容を比較し、以下の差異を検出する:

### 検出する差異

- **未翻訳**: 英語版に存在するが日本語版にない段落
- **原文更新済み**: 日本語ファイル内の `{/* EN: ... */}` コメントの内容と、現在の英語版の対応箇所が異なる箇所
- **削除済みコンテンツ**: 日本語版にあるが英語版にない段落
- **見出し構造の差異**: 見出し（`#`, `##`, `###` など）の構造が異なる箇所

## 出力形式

差異の一覧を以下の形式で表示する:

```
## 差異レポート: [ファイルパス]

### 見出し構造の差異
- (差異があれば列挙)

### 原文更新済み（EN コメントと英語版の不一致）
- (該当箇所と、ENコメントの内容 vs 現在の英語版の内容)

### 未翻訳（英語版にあるが日本語版にない）
- (該当箇所)

### 削除済み（日本語版にあるが英語版にない）
- (該当箇所)

### 修正案
- 各差異に対する具体的な修正案を提示する
```

**重要**: 自動修正は行わない。差異の報告と修正案の提示のみ行う。
